Game Jam - Shapes [2016]

Click here to download the Unity project


Shapes,   is a platforming game I created for a game jam in Q4 of 2015.   The theme of the game jam was shapes.   I worked on this game with my friend Daniel Ashton.   This was the most ambitious Unity project I had worked on.   It was a very useful learning experience as I learned a lot about working in Unity.   I did all the scripting for the game and still use some of the scripts I created then in my current Unity projects.

Geometry Wars remake in C++ [2015]

Click here to download the project


For my first semester at University we were to remake a class arcade game in C++.   I decided to remake Geormetry Wars.   The game has a very basic collision system.   It only used radial collisions.   However,   this game gave me the basic knowledge to make the collision physics system for my tank game in the second semester.   My game and code scored an 80%,   highest in my cohort.

"Game contains some advanced features,   paralax scrolling in backgrounds.   Player rotations and movement is advanced and well done.   Code is commented well in places,   still room for improvement.   Enemy movement is good,   games gets progressively more difficult as time passes.   Nice use of sprites for data entry on player scores."   -   Jamie Stewart (Academic Course Leader of Games Programming at University of Gloucestershire)
